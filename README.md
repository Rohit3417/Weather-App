# Weather-App

A Java-based **Weather Application** with GUI that fetches real-time weather data using an API.

## Structure

```
Weather-App/
├── src/
│   ├── Main.java        # Entry point
│   ├── Weather.java     # Weather data model & API calls
│   ├── Location.java    # Location handling
│   ├── CityData.java    # City data management
│   └── UI.java          # Swing GUI
├── lib/
│   └── json-20231013.jar
└── GUIBuilder/          # Custom GUI components & fonts
```

## Features
- Real-time weather data via API
- Swing-based GUI with custom fonts
- City search functionality

## Requirements
Java 11+, internet connection
